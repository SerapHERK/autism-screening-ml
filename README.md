# Autism Screening ML Demo

This repository presents a machine learning-based autism screening study using the ASD Toddler Dataset.

## Project Objective

The objective of this project is to investigate the effectiveness of machine learning algorithms in identifying Autism Spectrum Disorder (ASD) traits from screening questionnaire data.

## Dataset

The study uses the ASD Toddler Dataset collected through the ASDTests mobile application developed by Dr. Fadi Fayez.

## Implemented Methods

- Data preprocessing
- Label Encoding
- Train-Test Split (Hold-Out)
- SMOTE Oversampling
- Feature Normalization (StandardScaler)
- K-Fold Cross Validation

## Machine Learning Models

- Logistic Regression
- Random Forest
- Naive Bayes
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)

## Evaluation Metrics

- Accuracy
- Precision
- Recall (Sensitivity)
- Specificity
- F1 Score
- ROC-AUC
